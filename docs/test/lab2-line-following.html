Change decision mask to be smaller vertically, larger horizontally
Our racing line does have a black background. But remember the floor (which is the background of the racing line mat), is white. When there is a large area of white floor inside the decision window, there would be problems
In linedetect.cpp LineDetect::binthresh, we change the decision window to be smaller vertically
// img_mask(cv::Rect(0, 0, w, 0.8 * h)) = 0; img_mask(cv::Rect(0, 0, w, 0.85 * h)) = 0;
